## Guitar Club

### Club Class

Self-learner teaching guitar class for the first time, predictably crashed. During club class, discovered basically everyone can't keep up with the curriculum, whether it's club instructor class or teaching class. So this time I picked a super simple song —— [《Embrace》Mayday](https://www.youtube.com/watch?v=lvDa3r1pNuE). This song only has four chords from start to finish, except for $D_m7$, no finger needs to press two strings. The intro is also just T121 — basically a very suitable song for beginners.

Then an accident happened — because this song is too simple. Took about half an hour to teach, including waiting for people and playing the original. But can't end a two-hour class in half an hour, so I started teaching variations — muted strumming, percussive hitting, taught those, still only fifteen minutes passed. Had them play twice more, then couldn't think of anything else. So started the Q&A session. But this song is so simple, there won't be any questions. So everyone just chat until nine, dismiss!

Next club class to teach harder songs, like: [Planets](https://www.youtube.com/watch?v=Zn8M6mcGNoo) and [Stars](https://www.youtube.com/watch?v=l--9Ru69jTY) and [Star River](https://www.youtube.com/watch?v=RTUwaCImChM). Haven't decided yet, still have some time.

### McBopomofo Input

For Linux Chinese input methods, basically two choices. One is [New Chewying](https://chewing.im/), one is [McBopomofo](https://mcbopomofo.openvanilla.org/). At first thought, keyboard changed, might as well change the zhuyin input. After changing, found it's quite usable. Has customizable dictionary, commonly used characters can be manually added (while typing this article, dictionary added a few more characters). Don't need to select when typing that character next time. New Chewying goodbye, using McBopomofo.
