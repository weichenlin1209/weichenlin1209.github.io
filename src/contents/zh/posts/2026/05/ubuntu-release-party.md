# Ubuntu Release Party 2026
資訊，是一個被 0 跟 1 所組成的世界。你是否好奇：  

為什麼有人可以把 PC 遊戲帶到掌機上？  

為什麼 Steam Deck 不是安裝 Windows？  
而是選擇基於 Arch  Linux 為基礎開發成 SteamOS ？  

而最近當紅的「養龍蝦」── 讓 AI agent 在自己的電腦或雲端上執行任務？  
為什麼許多教學都推薦從 Ubuntu、Debian 、MacOs 這些類 Unix 作業系統開始？  

Linux ，也可以是日常使用、創作、遊戲、學習，甚至是部署 AI 工具的起點，他的迷人之處，在於它提供我們更多掌控電腦的自由。
從桌面長什麼樣子、快捷鍵、輸入法，到軟體套件與系統設定，都能一步步改造成真正適合自己直覺的工作環境。
這場活動不需要技術背景，解答你對日常上網、文書、電腦卡頓有關的所有問題。
歡迎一起來看看，讓電腦更像是自己的工具，使用起來變得更簡單！

今天，我們來一個 party ，來場 Linux 的盛宴吧！
讓這場分享與討論，變成屬於我們的派對！
